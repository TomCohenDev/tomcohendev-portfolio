## 🎨 Style Modifiers for Different Tools

### For DALL-E 3:

- Add: "Professional case study style, high quality, clean composition"
- Specify: "Side-by-side layout" or "Split screen comparison"

### For Midjourney:

- Add: `--ar 16:9` (for wide format) or `--ar 1:1` (for square)
- Add: `--style raw` (for more realistic) or `--style 4.0` (for artistic)
- Add: `--v 6` (for latest version)

### For Stable Diffusion:

- Add: "professional photography, high quality, detailed"
- Use negative prompts: "blurry, low quality, distorted, text errors"

### For Adobe Firefly:

- Specify: "Professional business illustration style"
- Use: "Before and after comparison" as main descriptor

## 📐 Recommended Dimensions

- **Case Study Header:** 1200x630px (16:9 ratio)
- **Blog Post Image:** 1200x800px (3:2 ratio)
- **Social Media:** 1200x627px (LinkedIn), 1080x1080px (Instagram)
- **Infographic:** 1920x1080px (Full HD)

## 🎯 Usage Tips

1. **Iterate:** Generate multiple variations and choose the best
2. **Refine:** Use the generated image as a base and refine in design tools
3. **Brand Colors:** Edit in Figma/Photoshop to match exact brand colors (purple theme)
4. **Add Text:** Overlay "BEFORE" and "AFTER" labels in design software
5. **Consistency:** Use similar style across all three before/after images
