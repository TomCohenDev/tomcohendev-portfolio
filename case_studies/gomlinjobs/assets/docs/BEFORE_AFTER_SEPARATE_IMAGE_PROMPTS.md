# Before/After Separate Image Generation Prompts

## Overview

Generate 4 separate images to replace the combined before/after comparison. Each image should be standalone and visually consistent with the dark mode, purple-accented aesthetic.

## Design Consistency Guidelines

All images should share:

- **Color Palette:**

  - Background: #1D201F (Dark Gray-Green)
  - Primary Accent: #bf95f9 (Purple)
  - Text: #E0D0C1 (Warm Off-White)
  - Secondary Text: #B9BBBE (Muted Gray)

- **Visual Treatment:**

  - "Before" images: Slightly desaturated, muted, showing inefficiency
  - "After" images: Vibrant, glowing purple accents, modern, clean
  - Consistent typography style
  - Glassmorphism effects where appropriate
  - Purple glow shadows on interactive elements (after images)

- **Quality:**
  - High resolution (1920x1080 minimum)
  - Realistic UI screenshots
  - Professional, polished appearance
  - No watermarks or placeholders
